# Sedano Brand — Sistema de Posts Instagram

Sistema visual completo para criar posts, carrosseis e stories no estilo Sedano Brand.

## Como usar

Abra o arquivo `index.html` diretamente no navegador.

Nenhuma instalação necessária — tudo roda localmente com React via CDN.

## Funcionalidades

- **Carrossel** — 10 slides com estrutura narrativa completa (Capa → Dor → Causa → Paradigma → Solução → Sistemas → Transformação → CTA)
- **Post Simples** — 8 templates prontos (quote, checklist, contraste, minimalista, híbrido, manifesto, caos/clareza, tabela)
- **Stories** — 4 formatos (gancho, dica, CTA, quote)
- **Geração via IA** — conteúdo gerado automaticamente por tema usando Claude
- **Editor inline** — edite qualquer campo diretamente no painel lateral
- **Legenda + Hashtags** — gerados junto com o carrossel

## Templates disponíveis

| Template | Uso |
|---|---|
| `capa` | Slide de abertura com headline grande |
| `interna` | Slides de conteúdo com corpo e destaque itálico |
| `contraste` | Antes vs. Depois lado a lado |
| `ondas` | Slide de solução com SVG animado |
| `diagrama` | Framework em passos numerados |
| `checklist` | Lista com ícones de check |
| `quote` | Citação autoral centralizada |
| `cta` | Chamada para ação com avatar |
| `minimalista` | Fundo branco, tipografia editorial |
| `hibrido` | Texto + placeholder de foto de estúdio |
| `manifesto` | Declaração centralizada de posicionamento |
| `chaos_clarity` | Duas palavras divididas por linha vertical |
| `tabela` | Comparativo em tabela |

## Identidade visual

- **Fontes:** Cormorant Garamond (headlines) + Inter (corpo)
- **Paleta base:** Preto `#000` + Branco + Neon `#00FFB2`
- **Cor de destaque:** configurável via painel Tweaks
- **Formato:** 1080×1080px (post) · 1080×1920px (story)

## Atalhos

- `← →` — navegar entre slides
- `Enter` — gerar conteúdo
