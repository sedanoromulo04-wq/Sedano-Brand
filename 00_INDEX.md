## LÓGICA DE DECISÃO PARA CRIAÇÃO

```
USUÁRIO DÁ UMA IDEIA
        ↓
[1] Ler brand/01 + brand/02 → entender identidade visual
        ↓
[2] Ler content/01 → montar roteiro 10 slides (prisão → liberdade)
        ↓
[3] Para cada slide → consultar brand/02 → escolher template correto
        ↓
[4] Criar design no Canva usando brand kit ID: kAGjljvyjpU
    Cores: APENAS #000000 / #FFFFFF / #DFFF00 / #1A1A1A
        ↓
[5] Gerar legenda + hashtags alinhados ao tom de voz
        ↓
[6] Filtrar pelo quality/01 → rejeitar se nível < 4 na matriz de profundidade
        ↓
ENTREGAR para aprovação do usuário
```

---

## REGRAS CRÍTICAS (nunca viole)

1. **Cores:** APENAS preto `#000000`, branco `#FFFFFF` e verde neon `#DFFF00`. Nenhuma outra cor.
2. **Brand kit Canva:** ID `kAGjljvyjpU` — sempre aplicar nas gerações.
3. **Profundidade mínima:** Nível 4 (Framework Proprietário) na matriz de profundidade técnica.
4. **Estrutura narrativa:** Todo carrossel segue prisão → liberdade (Content GPS).
5. **Qualidade visual:** Grid 1px presente, tipografia hierárquica, orbes neon para profundidade.
6. **Sem genérico:** Conteúdo clichê, cores erradas ou layout desorganizado = rejeitar e refazer.
