# Como Replicar o Template Matt Gray no Canva

Siga estes passos exatos para criar o design profissional que gerei para você:

---

## 1. Configuração do Documento
*   **Tamanho:** 1080 x 1350 px (Retrato do Instagram / 4:5). Este é o formato que o Matt Gray usa para ocupar mais espaço na tela.
*   **Fundo:** Altere a cor do fundo para Preto Puro (`#000000`).

---

## 2. Adicionando os Elementos Visuais
### O Orbe Neon (Glow)
1.  Vá em **Elementos** e pesquise por "Blur" ou "Gradient Circle".
2.  Escolha um círculo com bordas bem suaves.
3.  Mude a cor para Verde Lima/Neon (`#DFFF00`).
4.  Posicione no centro e aumente o tamanho.
5.  Reduza a **Transparência** para cerca de 40-60% para um efeito sutil.

### As Linhas de Grade (Grid)
1.  Vá em **Elementos** e pesquise por "Grid Lines" ou "Technical Lines".
2.  Procure por linhas finas brancas.
3.  Dica: Você também pode usar a tecla `L` para criar linhas individuais, mudar a espessura para `1` e a cor para Branco.
4.  Organize-as de forma assimétrica para criar um visual de "blueprint".

---

## 3. Tipografia (O Segredo)
### Título Principal (Serif)
*   **Fonte:** Pesquise no Canva por *Cormorant Garamond*, *Playfair Display* ou *Cinzel*.
*   **Estilo:** Tudo em maiúsculas (Uppercase), Negrito, cor Branca.
*   **Espaçamento:** Reduza um pouco o espaçamento entre linhas.

### Subtítulo (Sans-Serif)
*   **Fonte:** Pesquise por *Inter*, *Montserrat* ou *League Spartan*.
*   **Estilo:** Tudo em maiúsculas, cor Branca.
*   **Espaçamento:** Aumente o espaçamento entre letras (Letter Spacing) para dar um ar mais "premium".

---

## 4. Toque Final de Designer
*   **Contraste:** Certifique-se de que o texto está perfeitamente centralizado.
*   **Hierarquia:** O título deve ser pelo menos 3x maior que o subtítulo.
*   **Espaço Negativo:** Não tente preencher todos os cantos. O estilo do Matt Gray respira muito.

---

**Dica Pro:** Salve essas fontes e cores no seu "Brand Kit" do Canva para que todos os seus posts futuros tenham a mesma cara automaticamente!
