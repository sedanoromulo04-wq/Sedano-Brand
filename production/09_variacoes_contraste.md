# Guia: Novas Variações de Contraste (Estilo Matt Gray)

O contraste é a ferramenta mais poderosa para mostrar a transformação que você oferece. Aqui está como montar as 4 novas variações no Canva:

---

## 1. O Contraste Horizontal (Cima vs. Baixo)
*   **Conceito:** Divide a tela ao meio horizontalmente. Ideal para mostrar "O Jeito Antigo" vs. "O Novo Sistema".
*   **Como fazer:** Use uma linha de 1px para dividir a tela. Use **Cormorant Garamond** para os títulos e **Inter** para as listas.
*   **Dica:** O título de baixo deve ser em **Verde Neon** para simbolizar a solução.

## 2. O Contraste de Diagrama (Amador vs. Pro)
*   **Conceito:** Usa formas geométricas para representar a desorganização vs. a ordem.
*   **Como fazer:** Pesquise por "Abstract dots" para o lado amador e "Geometric pattern" ou "Sacred geometry" para o lado Pro.
*   **Dica:** Aplique um efeito de "Glow" (Brilho) verde neon no diagrama do lado Pro.

## 3. O Contraste de Foco (Caos vs. Clareza)
*   **Conceito:** Usa o desfoque para simbolizar a confusão e a nitidez para a solução.
*   **Como fazer:** No Canva, duplique seu texto. No primeiro, vá em "Efeitos" > "Desfoque" (Blur). No segundo, mantenha-o nítido e em verde neon.
*   **Dica:** Use uma linha vertical brilhante no centro para separar os dois estados.

## 4. O Contraste de Tabela Minimalista
*   **Conceito:** Uma comparação direta e técnica de pilares de negócio.
*   **Como fazer:** Use o elemento "Tabela" do Canva, mas remova o preenchimento das células, deixando apenas as bordas de 1px brancas.
*   **Dica:** Mantenha os textos alinhados ao centro para um visual mais equilibrado e autoritário.

---

**Dica Pro:** Sempre que usar o Verde Neon (`#DFFF00`), certifique-se de que ele seja a única cor vibrante na página. Isso garante que o olhar do leitor vá exatamente para onde você quer: a solução.
