# Skill: Branding "Designer/Founder" (Estilo Matt Gray)

Esta skill oferece um framework detalhado para construir uma identidade de marca digital que transmite autoridade, sofisticação e clareza, inspirada no estilo de "Designer/Founder" popularizado por Matt Gray.

## Visão Geral do Estilo

O branding "Designer/Founder" é caracterizado por:

*   **Minimalismo Luxuoso:** Uso estratégico de cores neutras (preto, branco, cinza) com um toque de cor vibrante (verde neon).
*   **Tipografia Elegante:** Combinação de fontes serifadas para títulos (autoridade) e sans-serif para corpo de texto (clareza).
*   **Conteúdo Sistemático:** Foco em frameworks, checklists e diagramas que simplificam conceitos complexos.
*   **Estética Cinematográfica:** Em vídeos, iluminação controlada, cortes rápidos e legendas dinâmicas.

## Como Usar Esta Skill

Esta skill é dividida em seções para facilitar a aplicação dos conceitos:

### 1. Identidade Visual Base

Para configurar a base do seu branding, consulte:
*   **Paleta de Cores:** Preto (`#000000`), Branco (`#FFFFFF`), Verde Neon (`#DFFF00`), Cinza Escuro (`#1A1A1A`).
*   **Tipografia:** Fontes Serif (ex: Cormorant Garamond) e Sans-Serif (ex: Inter).

Para detalhes sobre como configurar seu Brand Kit no Canva, consulte o arquivo `references/brand_kit_canva.md`.

### 2. Modelos de Posts para Canva

Explore os diversos modelos de posts para Instagram, cada um com um propósito específico:

*   **Carrosséis:** Capas de alto impacto e páginas internas com estrutura de valor.
*   **Checklists:** Listas rápidas para gerar salvamentos.
*   **Diagramas:** Visualização de sistemas e processos.
*   **Quote Cards:** Frases de impacto para compartilhamento.
*   **Posts de Contraste:** Comparativos "Antes vs. Depois" ou "Problema vs. Solução".
*   **Gráficos de Ondas:** Representação visual de conceitos abstratos e progresso.

Cada modelo possui um guia detalhado de montagem no Canva. Para acessar as instruções e exemplos visuais, consulte o arquivo `references/canva_templates.md`.

### 3. Modelo de Vídeo (Reels/Shorts)

Para criar vídeos dinâmicos e engajadores no estilo Matt Gray, que incluem:

*   **Roteiro Estratégico:** Gancho, Conteúdo e CTA.
*   **Edição Dinâmica:** Cortes rápidos, zooms sutis e B-roll relevante.
*   **Legendas Dinâmicas:** Texto branco com palavras-chave em Verde Neon.
*   **Áudio e Ambiente:** Trilha sonora minimalista e iluminação de estúdio.

Para um guia completo sobre como editar seus vídeos, consulte o arquivo `references/video_style.md`.
