# Guia de Clonagem de Brand: Estilo Matt Gray (Founder OS)

Este guia detalha os elementos visuais, estratégicos e de conteúdo utilizados por **Matt Gray** (@matthgray) para que você possa replicar essa estética de "Designer/Founder" de alta performance em seu próprio perfil.

---

## 1. Identidade Visual (The Aesthetic)

O estilo de Matt Gray é uma mistura de **minimalismo luxuoso** com **tecnologia moderna**. É focado em clareza, autoridade e sistemas.

### Paleta de Cores
A paleta é extremamente restrita para manter a consistência e o reconhecimento imediato.

| Elemento | Cor | Hex Code |
| :--- | :--- | :--- |
| **Fundo Principal** | Preto Profundo | `#000000` |
| **Texto/Destaque** | Branco Puro | `#FFFFFF` |
| **Cor de Sotaque** | Verde Neon / Lima | `#DFFF00` ou `#CBFD03` |
| **Secundária** | Cinza Escuro | `#1A1A1A` |

### Tipografia
O segredo do design dele está no contraste entre uma fonte **Serifada elegante** e uma **Sans-Serif técnica**.

*   **Títulos (Serif):** Use fontes como *Cormorant Garamond*, *Playfair Display* ou *Chronicle Display*. Transmite autoridade e sofisticação.
*   **Corpo e Dados (Sans-Serif):** Use fontes como *Inter*, *Satoshi*, *Montserrat* ou *Helvetica Now*. Transmite modernidade e clareza.

## 2. Tipos de Postagens (Content Pillars)

Matt Gray utiliza formatos específicos que "viciam" o algoritmo e entregam valor rápido.

### A. O Carrossel de Framework (O Carro-Chefe)
*   **Capa:** Título grande em Serif, subtítulo em Sans-Serif, e um elemento visual (como um orbe brilhante ou ícone 3D).
*   **Conteúdo:** Passo a passo de um sistema (ex: "Como construir uma audiência em 30 dias").
*   **Design:** Fundo preto, linhas finas brancas separando seções, uso pontual do verde neon para destacar palavras-chave.

### B. O Checklist do Fundador
*   **Formato:** Uma lista simples de itens com "checkmarks" em verde neon.
*   **Objetivo:** Gerar salvamentos. As pessoas amam listas acionáveis.

### C. Quote Cards de Alto Impacto
*   **Design:** Apenas texto branco centralizado em fundo preto. Às vezes com o nome do perfil em tamanho pequeno no rodapé.
*   **Conteúdo:** Frases curtas sobre mentalidade, sistemas ou negócios.

### D. Reels de "Talking Head" com Legendas Dinâmicas
*   **Visual:** Matt falando diretamente para a câmera, fundo desfocado ou em um escritório minimalista.
*   **Legendas:** Estilo "Alex Hormozi", mas mais limpo. Palavras mudando de cor para o verde neon conforme ele fala.

---

## 3. Elementos de Design (The Designer Touch)

Para "clonar" o visual, você deve incorporar estes elementos gráficos:

1.  **Orbes e Gradientes:** Use círculos com desfoque gaussiano alto (Gaussian Blur) na cor verde neon sobre o fundo preto para criar profundidade.
2.  **Linhas de Grade (Grid Lines):** Use linhas brancas de 1px de espessura para criar uma estrutura de "blueprint" ou sistema.
3.  **Fotografia de Contraste:** Fotos em preto e branco com alto contraste ou fotos coloridas com tons frios e sombras profundas.
4.  **Ícones Minimalistas:** Use ícones de linha fina (Linear Icons) em vez de ícones preenchidos.

---

## 4. Estratégia de Conteúdo (The System)

Matt não posta apenas por postar; ele segue o **"Content GPS"**:

*   **Gancho (Hook):** Os primeiros 3 segundos do Reel ou a primeira página do carrossel devem prometer um resultado específico (ex: "Como eu escalo marcas").
*   **Retenção:** Use listas e diagramas visuais para manter o usuário interessado.
*   **CTA (Chamada para Ação):** Sempre peça um comentário específico para enviar um link (ex: "Comente 'BRAND' para receber meu template").

---

## Como Começar (Checklist de Implementação)

1. [ ] Defina sua fonte Serif e Sans-Serif.
2. [ ] Crie um template de carrossel no Canva ou Figma usando fundo `#000000` e detalhes em `#DFFF00`.
3. [ ] Tire 5-10 fotos profissionais com roupas neutras (preto/branco/cinza).
4. [ ] Escreva 3 "Frameworks" que você domina e transforme-os em carrosséis.
5. [ ] Use o verde neon apenas para o que for **mais importante** na tela.
